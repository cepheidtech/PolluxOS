# PolluxOS

![PolluxOS](/readme_assets/readme_thumb_new.png)

Welcome! This is the repository for PolluxOS, a series of [TurboWarp](https://github.com/TurboWarp/) "operating system" projects made by [SabirICT](https://scratch.mit.edu/users/SabirICT) (also known as [cepheid.tech](https://scratch.mit.edu/users/cepheidtech/)) between 2023 and 2026.

## Getting around

The .sb3 files for each public version of the OS can be found in `/sb3/v1x`.
